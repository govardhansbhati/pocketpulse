Theme Colors
Primary Theme Colors

Primary:
Hex: #4CAF50
RGB: (76, 175, 80)
Description: A calming green to represent growth and financial stability. It gives a positive vibe for money-related apps.
Secondary:
Hex: #FF9800
RGB: (255, 152, 0)
Description: A vibrant orange to add energy and highlight actionable items, like buttons for "Add Spending" or "Received Money."
Accent:
Hex: #673AB7
RGB: (103, 58, 183)
Description: A deep purple to create a premium, sophisticated look for highlights like charts or key statistics.
Neutral Colors (for Text and Background)

Background:
Hex: #F9FAFB
RGB: (249, 250, 251)
Description: A very light gray or white background to keep it clean and minimal.
Text (Primary):
Hex: #212121
RGB: (33, 33, 33)
Description: A dark, nearly black color for main text. It ensures readability and contrasts well with the light background.
Text (Secondary):
Hex: #757575
RGB: (117, 117, 117)
Description: A gray tone for secondary text like hints or small labels.
Fonts
Primary Font: "Inter"

Description: A modern, sans-serif font designed for readability and clarity. It’s ideal for apps where the focus is on numbers, charts, and lists.
Font Weights:
Regular (400): For body text and transaction descriptions.
Medium (500): For slightly emphasized text, like balances or section headers.
Bold (700): For key numbers like available balance or total income/expense.
Alternative Font: "SF Pro" (Native iOS Font)

Description: Apple's default system font. It ensures a seamless look across iOS apps and is optimized for legibility.
Font Weights:
Regular: For body text.
Semibold/Bold: For headings and highlighted stats.
Typography Recommendations:

Header (H1): Available Balance
Font: Inter Bold (700)
Size: 32px
Color: #212121 (Primary Text Color)
Subheading (H2): Income/Expense
Font: Inter Medium (500)
Size: 20px
Color: #4CAF50 (Income) / #FF9800 (Expense)
Body Text:
Font: Inter Regular (400)
Size: 16px
Color: #212121
Secondary Text (Hints/Labels):
Font: Inter Regular (400)
Size: 14px
Color: #757575
Buttons:
Font: Inter Medium (500)
Size: 18px
Color: White (#FFFFFF) on Primary (#4CAF50) or Secondary (#FF9800) background.
